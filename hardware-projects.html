Mortal Mayhem

2-player handheld game. Emulation of the retro classic Mortal Kombat

This 2-player handheld game is powered by the TI TIVA TM4C123G lauchpad. Custom-built hardware includes buttons for menu item selection/ attack, DACs for sound output, ADCs for joystick sampling, wireless receiver and transmitter antennas for serialized player-to-player communication, and an LCD screen for display. The main game routine is programmed in C. C++ modules are used to initialize hardware and communication protocols. Hardware interrupts are triggered on player input and are serviced using interrupt service routines. Busy-Wait threads and sempahores are used for synchronizig interrupts. The Bounding Volume Hierarchy (BVH) algorithm is used for collision detection, which models gameplay events where one player attacks another. The algorithm encloses the sprites into geometric shapes and assesses overlap. Race conditions are caused by 2 players striking at the same time, which are solved by a randomization algorithm that arbitrarily sets priority. An SPI communication module is implemented for communication with LCD screen. Serialization is configured using queues for FIFO data retrieval. Error codes are generated for error-checking. Images are stored as hexadecimal pixel matrices, and are loaded via simple array parsing into buffers for display. Sound/ music is compressed using WAV files.

Elevator Motor Bridge Driver

Printed Circuit Board

PCB
Architecture
MOSFETs in H-Bridge

Board powering a DC motor from a solar panel, competing with other boards to deliver the maximum power to the motor. Voltage dividers and current sensing resistors measure the solar panel's voltage (V_solar) and current (I_solar), with Op-Amps conditioning these signals. Custom ADC implemented to allow microcontroller to read the signals and run the Perturb & Observe algorithm to determine the optimal operating point of the Photovoltaic System. The microcontroller dynamically configures a programmable gain amplifier (PGA) to adapt to varying signal levels, ensuring accurate measurements. The control signal is output to a DAC, which converts it to an analog voltage for a voltage-controlled PWM generator. This PWM signal controls two MOSFETs in a half-bridge configuration, with an additional analog feedback loop using a precision voltage reference and a low-pass filter to stabilize and fine-tune the PWM signal for more efficient power delivery. The PCB is designed and fabricated using KiCAD, correct placement and routing. Testing involves verifying signal measurements, microcontroller functionality, PWM output, and overall system performance, under varying solar conditions to ensure maximum power delivery.
